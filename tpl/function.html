History API(redirect without refresh)

html4 history api:
1. history.go(0) = location.reload() 
   -go(x) means goto the page |x| pages before/after
2. history.back() = history.go(-1) 
   -go back to last page
3. history.foward() = history.go(1) 
   -go foward 1 step
4. history.length 
   -the history pages stack length
html5 history api:
1. history.pushState(data, title, '' || url )  
   -data: needs to transport; 
   -title: the new page title; 
   -url: the new page url,'' means current page;
   -example: 
	history.pushState({ foo: "bar" }, "bar page", "bar.html");
	history.pushState({page: 1}, "title 1", "?page=1");
