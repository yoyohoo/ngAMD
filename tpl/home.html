{{ title }}

The view(index.html):

<html ng-app="uiApp">
  <head>
    <script src="js/lib/angular.min.js"></script>
    <script src="js/lib/angular-route.min.js"></script>
  </head>
  <body>
    <div class="container">
      <div ng-view></div>
    </div>
  </body>
</html>

The template(home.html):

<div class="col-md-12">
  <div class="page-header">
    <h3>{ { title } }</h3>
  </div>
</div>

The router(app.js):

angular.module('uiApp', ['ngRoute','uiControllers'])
    .config(['$routeProvider', function($routeProvider) {
	 $routeProvider.when("/Home", {
	     templateUrl: 'tpl/Home.html',
	     controller: 'HomeController'
	 })
    }])
})(window.angular);

The controller(uiController.js):

(function(ng) {
    ng.module('uiControllers', [])
    .controller('HomeController', ['$scope', function($scope) {
        $scope.title = 'Simple start:';
    }])
})(window.angular);

The directive: ng-app in the root element marks the page as an angularjs application, and leads this application to bootstrap from which module.

The directive: ng-view in the body helps to include the template rendered by the current route(from thelocation hash) into the main layout view, it changes whenever the router changed.

The $routeProvider binding the controller and views, it needs the module ngRoute to be installed.

{ { title } } binding the value from $scope.title = 'Simple start:';
